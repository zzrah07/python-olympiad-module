Olympiad Module

Description

Olympiad Module is a quiz application developed using Python and Tkinter. This project was created during my 12th grade as part of my computer science learning. It allows teachers to create quiz questions and students to attempt them through a simple graphical interface.

Features

- Teacher interface to add and manage quiz questions
- Supports Multiple Choice Questions (MCQ), True/False, and Fill in the Blanks
- Students can attend the quiz by entering their name
- Stores questions and student answers using Python file handling
- Teachers can view the results of each student

Technologies Used

- Python
- Tkinter (GUI Library)
- File Handling for storing questions and responses

How to Run the Project

1. Install Python on your system
2. Clone or download this repository
3. Run the main Python file:

python main.py

Project Purpose

This project was developed during my 12th standard to learn Python programming concepts such as GUI development using Tkinter and file handling for data storage.
